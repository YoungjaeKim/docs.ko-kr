# <a name="compiler-error-cs0404"></a>컴파일러 오류 CS0404
예기치 않은 '<'입니다. 특성은 제네릭이 될 수 없습니다.  
  
 제네릭 형식 매개 변수가 특성에서 허용되지 않습니다. 형식 매개 변수 및 꺾쇠 괄호를 제거합니다.  
  
 다음 샘플에서는 CS0404를 생성합니다.  
  
```csharp  
// CS0404.cs  
[MyAttrib<int>]  // CS0404  
class C  
{  
   public static void Main()  
   {  
  
   }  
}  
```

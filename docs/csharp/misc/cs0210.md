# <a name="compiler-error-cs0210"></a>컴파일러 오류 CS0210
fixed 또는 using 문 선언에 이니셜라이저를 입력해야 합니다.  
  
 [fixed 문](../../csharp/language-reference/keywords/fixed-statement.md)에서 변수를 선언하고 초기화해야 합니다. 자세한 내용은 [안전하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md)를 참조하세요.  
  
 다음 샘플에서는 CS0210을 생성합니다.  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 또한 다음 샘플에서는 [using 문](../../csharp/language-reference/keywords/using-statement.md) 에 이니셜라이저가 없기 때문에 CS0210을 생성합니다.  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```

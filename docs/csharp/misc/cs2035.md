# <a name="compiler-error-cs2035"></a>컴파일러 오류 CS2035
명령줄 구문 오류: 누락 된 ':\<번호 >' 'compiler_option' 옵션에 대 한  
  
 일부 컴파일러 옵션은 값이 필요합니다.  
  
## <a name="example"></a>예  
 다음 샘플에서는 CS2035를 생성합니다.  
  
```csharp  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```

# <a name="compiler-error-cs1586"></a>컴파일러 오류 CS1586
배열을 만들 때에는 배열 크기 또는 배열 이니셜라이저가 있어야 합니다.  
  
 배열이 잘못 선언되었습니다.  
  
 다음 샘플에서는 CS1586을 생성합니다.  
  
```csharp  
// CS1586.cs  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      int[] a = new int[];   // CS1586  
      // try the following line instead  
      int[] b = new int[5];  
   }  
}  
```

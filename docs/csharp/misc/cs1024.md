# <a name="compiler-error-cs1024"></a>컴파일러 오류 CS1024
전처리기 지시문이 필요합니다.  
  
 줄이 파운드 기호(#)로 시작하지만 후속 문자열이 유효한 [전처리기 지시문](../../csharp/language-reference/preprocessor-directives/index.md)이 아니었습니다.  
  
 다음 샘플에서는 CS1024를 생성합니다.  
  
```csharp  
// CS1024.cs  
#import System   // CS1024  
```

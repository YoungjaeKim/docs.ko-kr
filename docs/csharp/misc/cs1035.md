# <a name="compiler-error-cs1035"></a>컴파일러 오류 CS1035
파일 끝(EOF)이 있습니다. '*/'가 필요합니다.  
  
 여는 주석 구분 기호와 닫는 구분 기호의 짝이 맞지 않습니다.  
  
 다음 샘플에서는 CS1035를 생성합니다.  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
